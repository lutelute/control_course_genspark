# 制御理論講義 (GenSpark版)

## 概要
本プロジェクトは制御理論の包括的な講義資料を提供します。基本的な状態空間表現から最適制御理論まで、16回の講義で現代制御理論の基礎を学習できます。

## プロジェクト構造

```
control_course_genspark/
├── README.md                    # このファイル
├── index.html                   # メインページ（講義一覧）
├── html/                        # HTML形式の講義資料
│   ├── 第01回_システムの状態空間表現.html
│   ├── 第02回_PID制御.html
│   ├── ...
│   └── 第16回_まとめ.html
└── markdown/                    # Markdown形式の講義資料
    ├── 第01回_システムの状態空間表現.md
    ├── 第02回_PID制御.md
    ├── ...
    └── 第16回_まとめ.md
```

## 講義内容

### 基礎編 (第1-5回)
- 第01回: システムの状態空間表現
- 第02回: PID制御
- 第03回: 状態方程式の解
- 第04回: 遷移行列の計算
- 第05回: 伝達関数と状態空間表現

### 解析編 (第6-9回)
- 第06回: 安定性と極
- 第07回: リアプノフ方程式
- 第08回: 可制御性
- 第09回: 可観測性

### 制御設計編 (第10-12回)
- 第10回: 状態フィードバックと極配置
- 第11回: オブザーバ
- 第12回: 最適レギュレータ

### 応用編 (第13-16回)
- 第13回: リッカチ方程式と最適性
- 第14回: カルマンフィルタ
- 第15回: サーボシステム
- 第16回: まとめ

## 使用方法

### ウェブブラウザでの閲覧
1. `index.html`をウェブブラウザで開く
2. 各講義のHTMLリンクをクリックして内容を閲覧

### ローカル環境での編集
1. Markdownファイルを編集
2. 必要に応じてHTMLファイルを更新
